This repository serves as the master repository for all tasks completed during the DecodeLabs Backend Development Internship Program. Each task demonstrates different backend development concepts and skills, ranging from REST API fundamentals to third-party API integration.

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* Axios
* REST APIs
* Postman
* Environment Variables (.env)
